@font-family-serif:       'Lato', sans-serif;

    Get started with the LESS

    Short list of guide how to start to use LESS preprocessor.

    1. Download and install a program which will compile LESS files into simple CSS. We use for this task application Prepros (It's supported by OS X, Windows and Linux). Of course, you can use a task runner or plugin in your text editor as well.
    2. Set your project folder in the Prepros.
    3. Now you can start to customize LESS files and let Prepros to generate simple CSS files.

    Make your own theme

    You will learn how to make custom theme, you will have to create two files. Firstly for main-theme subfolder and secondly for bootstrap subfolder.

    Firstly for main theme

    1. Find HTML/assets/less/main-theme/themes folder and create there new file, let's call it new-theme-m.less. The -m suffix is mark for main-theme, -b is for bootstrap.
    2. Copy content below and insert its there. Choose your own colors, if you want. Now you can save it and compile it into simple CSS. Or you can continue to setting up of your new theme.
    3. // Main Colors
@color1:        @cyan500;
@color2:        @blueGrey900;
@color3:        @blueGrey800;
@color4:        @blueGrey50;
@color5:        @blueGrey200;
@color6:        @blueGrey100;
 
@primary:        @color1;
@success:        @color1;
@info:           @color1;
@warning:        @color1;
@danger:         @color1;
 
// Main Theme  
@import "../main-theme.less";

      These variable are from material design, there are all place in the "assets/less/main-theme/helpers/material-design.less" file

    5. Go to find assets/less/main-theme/main-theme.less file and choose which files you want include to new-theme-m.less and which you don't want to use. Remove them, save it and compile new-theme-m.less.
    6. In the index.html, replace old theme by the new one - assets/css/main-theme/themes/new-theme-m.css. (set your own path, it depends on your compiler settings).

    jQuery Knob

    Nice, downward compatible, touchable, jQuery dial.

    JS Installation

    <script src="assets/js/plugins/jquery.knob.js"></script>

    The code below is placed in HTML/assets/js/plugins-setup.js file.

     $(".lbKnob").knob({
        min: 0,
        fgColor: "#26C6DA", // SET YOUR COLOR
        max: 100,
        step: 5,
        angleOffset: 0,
        angleArc: 360,
        stopper: true,
        readOnly: true,
        cursor: false,
        lineCap: 'none',
        thickness: '0.03',
        width: 150,
        height: 150,
        displayInput: true,
        displayPrevious: true,
        inputColor: '#999999',
        font: 'Lato',
        fontWeight: 'normal',
        bgColor: '#ddd',
        draw: function() {
            if (this.$.data('skin') == 'tron') {
                var a = this.angle(this.cv), // Angle
                    sa = this.startAngle, // Previous start angle
                    sat = this.startAngle, // Start angle
                    ea, // Previous end angle
                    eat = sat + a, // End angle
                    r = 1;
                this.g.lineWidth = this.lineWidth;
                this.o.cursor && (sat = eat - 0.3) && (eat = eat + 0.3);
                if (this.o.displayPrevious) {
                    ea = this.startAngle + this.angle(this.v);
                    this.o.cursor && (sa = ea - 0.3) && (ea = ea + 0.3);
                    this.g.beginPath();
                    this.g.strokeStyle = this.pColor;
                    this.g.arc(this.xy, this.xy, this.radius - this.lineWidth, sa, ea, false);
                    this.g.stroke();
                }
                this.g.beginPath();
                this.g.strokeStyle = r ? this.o.fgColor : this.fgColor;
                this.g.arc(this.xy, this.xy, this.radius - this.lineWidth, sat, eat, false);
                this.g.stroke();
                this.g.lineWidth = 2;
                this.g.beginPath();
                this.g.strokeStyle = this.o.fgColor;
                this.g.arc(this.xy, this.xy, this.radius - this.lineWidth + 1 + this.lineWidth * 2 / 3, 0, 2 * Math.PI, false);
                this.g.stroke();
                return false;
            }
        }
    });

Change Style of Navigation

There are three style of navbar - dark = .navbar-inverse, light = .navbar-default and transparent = .navbar-trn

Find this (It's almost on the top in the index.html):

<div class="navbar navbar-trn navbar-fixed-top navbar-height" role="navigation" id="navbarSettings">

And change the navbar-trn for navbar-default or navbar-inverse.

Page Width

If you want wide option, just add this class wrap-width to the <body> tag.

Page Background

You have to create new class, let's called it wrap-background. And add it to the <body> tag.

.wrap-background {
    background: url('../../../img/patterns/crossword/crossword.png') //set your own path, base on location your image
    background-repeat: repeat;
    background-attachment: fixed !important;
}
